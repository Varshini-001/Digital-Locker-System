**Digital Locker System**
A secure web application to store, manage, and access personal documents digitally. Users can upload, view, download, and delete documents anytime from a centralized platform.

**Features**

* Upload documents securely
* View stored documents
* Download files anytime
* Delete unwanted documents
* Secure document handling

**Tech Stack**

* **Frontend:** React.js
* **Backend:** Spring Boot (Java)
* **Database:** (Add yours – MySQL / H2 / etc.)
* **Tools:** Git, GitHub, VS Code

**Project Structure**

Digital-Locker-System/
│── reactapp/     # Frontend (React)
│── springapp/    # Backend (Spring Boot)
│── README.md

**How to Run the Project**
**Step 1: Clone the Repository**
git clone https://github.com/Varshini-001/Digital-Locker-System.git
cd Digital-Locker-System

**Step 2: Run Backend (Spring Boot)**
cd springapp
mvn spring-boot:run

Runs on: http://localhost:8080

**Step 3: Run Frontend (React)**
Open a new terminal:
cd reactapp
npm install
npm start

Runs on: http://localhost:8081

**Conclusion**

This project demonstrates a full-stack application using React and Spring Boot to provide a secure and efficient digital document management system.
